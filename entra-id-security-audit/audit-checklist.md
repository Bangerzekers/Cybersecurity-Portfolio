# Checklist d’audit Entra ID

## Comptes et authentification

- [x] MFA activé pour les administrateurs
- [ ] MFA activé pour les utilisateurs
- [ ] Authentification héritée bloquée
- [ ] Comptes inactifs identifiés
- [ ] Comptes partagés recensés
- [ ] Comptes invités révisés

## Privilèges

- [ ] Nombre de Global Administrators limité
- [ ] Comptes administratifs séparés
- [ ] Attribution des rôles justifiée
- [ ] Utilisation de PIM évaluée
- [ ] Comptes d’urgence documentés et testés

## Accès conditionnel

- [ ] Politique MFA
- [ ] Blocage des pays non autorisés, si pertinent
- [ ] Protection des comptes privilégiés
- [ ] Contrôle des appareils
- [ ] Exclusions documentées
- [ ] Mode report-only testé avant activation

## Applications

- [ ] Consentements administrateurs revus
- [ ] Applications inutilisées supprimées
- [ ] Secrets et certificats expirés vérifiés
- [ ] Permissions à haut risque identifiées

## Gouvernance

- [ ] Revue trimestrielle des accès
- [ ] Journalisation disponible
- [ ] Alertes sur les changements de rôles
- [ ] Procédure de départ collaborateur
- [ ] Plan de remédiation suivi
