# Politique de sécurité

Ce dépôt contient uniquement des démonstrations réalisées en laboratoire.

Ne jamais publier :

- mots de passe ;
- jetons ;
- clés API ;
- certificats privés ;
- noms réels d’utilisateurs ;
- adresses IP internes d’une organisation ;
- journaux contenant des données personnelles ;
- informations provenant d’un employeur ou d’un client.

Pour signaler un problème dans ce dépôt, utiliser un canal privé plutôt qu’une issue publique.
